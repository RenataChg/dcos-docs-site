# Description
The `dcos auth` command authenticates a user to the DC/OS cluster.

# Usage

```bash
dcos auth <command> [OPTIONS]
```

# Options

| Name | Description |
|---------|-------------|-------------|
| `--help, h`   | Display usage. |
| `--info`   | Display a short description of this subcommand. |
| `--version`   | Display version information. |
